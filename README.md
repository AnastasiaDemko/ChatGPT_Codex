# Finance Balance Tracker (Android)

Приложение на Jetpack Compose для учёта личных финансов в **BYN**.

## Что реализовано
- Счета: создание, удаление, стартовый баланс, выбор иконки/цвета.
- Категории: создание, удаление, тип (доход/расход), выбор иконки/цвета.
- Операции: доход/расход с выбором счёта и категории, дата: сегодня/вчера/позавчера.
- Переводы между счетами.
- История: список операций, фильтры (день/неделя/месяц/всё время), круговая диаграмма расходов.
- Светлая тема Material 3.

## Технологии
- Kotlin
- Jetpack Compose
- Material 3

## Запуск
1. Открыть проект в Android Studio Iguana+.
2. Sync Gradle.
3. Запустить `app` на эмуляторе или устройстве.
